# Iris Dataset — Exploratory Data Analysis (EDA)

**Project:** Simple EDA of the Iris dataset  

## Overview
This repository contains a short exploratory data analysis (EDA) of the classic Iris dataset using Python. It demonstrates loading the dataset, inspecting structure, and creating basic visualizations:
- Scatter plots (sepal and petal relationships)
- Histogram (distribution of sepal length)
- Boxplots (spread and outliers detection)

## Files
- `iris_analysis.py` — main script. Loads dataset with seaborn and creates the plots using matplotlib.
- `.gitignore` — common files to ignore.
- `images/` — folder for saved plots.

## How to Run
To run the analysis, simply execute the Python script:

```bash
python iris_analysis.py
